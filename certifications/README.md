## 📚 Completed Coursework

### Theoretical Foundations
| Subject | Key Topics Covered |
|:---|:---|
| **Ethical Hacking** | Reconnaissance, Scanning, Exploitation, Post-exploitation, Reporting |
| **Cryptography** | Symmetric (AES, DES), Asymmetric (RSA, ECC), Hash Functions (SHA, MD5), PKI |
| **Steganography & Digital Watermarking** | LSB techniques, DCT embedding, Steganalysis, ARSS Algorithm (Arjuna's Astra) |
| **Mathematical Foundations for Security** | Number Theory, Modular Arithmetic, Group Theory, Probability for Cryptanalysis |
| **Operating Systems** | Process management, Memory security, File permissions, Kernel security |
| **Systems & Network Security** | TCP/IP stack, Firewalls, IDS/IPS, VPN, VLAN segmentation |
| **Cyber Forensics** | Chain of custody, Disk imaging, Artifact recovery, Timeline analysis |
| **Storage Management & Security** | RAID, SAN/NAS security, Encryption at rest, Access control |
| **Web Programming** | HTML/CSS/JS, HTTP security headers, CORS, Content Security Policy |
| **Intrusion Detection & Prevention** | Signature-based vs anomaly-based, Snort rules, SIEM concepts |
| **Mobile Computing Security** | Android security model, Mobile threats, App sandboxing |
| **Biometric Security** | Fingerprint, Iris, Face recognition systems, FAR/FRR metrics |
